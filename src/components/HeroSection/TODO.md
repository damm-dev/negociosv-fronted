# Tareas para embellecer HeroSection

## Plan general
Crear un diseño visual muy atractivo para la Hero Section con los siguientes puntos:

- Estilo visual y fondo:
  - Agregar un fondo degradado o imagen atractiva con buen contraste.
  - Aplicar un filtro o efecto suave al fondo para que el texto resalte.

- Contenedor Hero Content:
  - Limitar el ancho máximo para mejorar legibilidad.
  - Centrar vertical y horizontalmente el contenido.
  - Añadir padding y márgenes adecuados.

- Tipografía:
  - Usar fuentes atractivas y modernas.
  - Mejorar tamaño, color y peso de h1 y p.
  - Añadir sombra de texto para mejorar visibilidad.

- SearchBar:
  - Ajustar la ubicación y estilo para que armonice con la sección Hero.

- Animaciones sutiles:
  - Añadir animaciones de entrada para el texto y el search bar (fade-in/slide).
  - Suavizar transiciones y efectos.

- Responsividad:
  - Adaptar tamaño y disposición del contenido para pantallas móviles.
  - Ajustar tamaño de fuente y espacios.

## Pasos para implementación

- [ ] Crear estilos CSS para la sección Hero en HeroSection.css.
- [ ] Añadir animaciones CSS para fade-in en texto y SearchBar.
- [ ] Modificar HeroSection.jsx solo si es necesario para añadir wrappers de animación o estructura.
- [ ] Probar visualmente en distintos tamaños.
- [ ] Ajustar detalles y confirmar que solo se modifican HeroSection.jsx y HeroSection.css.
